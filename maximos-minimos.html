Conceptos de Superficies

Capítulo 5 Máximos, Mínimos y Optimización

🎧 Podcast del capítulo

Una de las aplicaciones más importantes del cálculo diferencial es la búsqueda de valores extremos de funciones. En este capítulo se desarrollan los criterios para identificar máximos y mínimos locales de funciones de dos y tres variables, utilizando el Hessiano y los multiplicadores de Lagrange.

5.1 Puntos críticos

5.1.1 Definición

Un punto \((a, b)\) es un punto crítico de \(f(x, y)\) si:

  1. \(\nabla f(a, b) = \vec{0}\), es decir, \(f_x(a, b) = 0\) y \(f_y(a, b) = 0\); o
  2. Alguna de las derivadas parciales no existe en \((a, b)\).

Los puntos críticos de tipo 1 se llaman puntos estacionarios.

5.1.2 Extremos locales

  • \(f(a, b)\) es un máximo local si \(f(x, y) \leq f(a, b)\) para todo \((x, y)\) en una vecindad de \((a, b)\).
  • \(f(a, b)\) es un mínimo local si \(f(x, y) \geq f(a, b)\) para todo \((x, y)\) en una vecindad de \((a, b)\).
  • Un punto crítico que no es ni máximo ni mínimo local se denomina punto de silla.

5.2 Criterio del Hessiano para funciones de dos variables

5.2.1 Matriz Hessiana

La matriz Hessiana de \(f\) en \((a, b)\) es:

\[ H(f) = \begin{pmatrix} f_{xx} & f_{xy} \\ f_{yx} & f_{yy} \end{pmatrix} \]

5.2.2 Discriminante

El discriminante (o determinante del Hessiano) es:

\[ D = f_{xx} f_{yy} - (f_{xy})^2 = \det(H) \]

5.2.3 Criterio de clasificación

Sea \((a, b)\) un punto crítico de \(f\) con segundas derivadas parciales continuas en una vecindad de \((a, b)\):

Condición Clasificación
\(D > 0\) y \(f_{xx} > 0\) Mínimo local
\(D > 0\) y \(f_{xx} < 0\) Máximo local
\(D < 0\) Punto de silla
\(D = 0\) Criterio inconcluso

5.2.4 Ejemplo resuelto

Ejemplo 1. Clasifique los puntos críticos de \(f(x, y) = x^3 + y^3 - 3xy\).

Solución. Las derivadas parciales son:

\[ f_x = 3x^2 - 3y = 0 \implies y = x^2 \]

\[ f_y = 3y^2 - 3x = 0 \implies x = y^2 \]

Sustituyendo \(y = x^2\) en la segunda ecuación: \(x = x^4\), es decir, \(x(x^3 - 1) = 0\).

  • Si \(x = 0\), entonces \(y = 0\). Punto crítico: \((0, 0)\).
  • Si \(x = 1\), entonces \(y = 1\). Punto crítico: \((1, 1)\).

Las segundas derivadas son \(f_{xx} = 6x\), \(f_{yy} = 6y\), \(f_{xy} = -3\).

En \((0, 0)\): \(f_{xx} = 0\), \(f_{yy} = 0\), \(f_{xy} = -3\).

\[ D = (0)(0) - (-3)^2 = -9 < 0 \implies \text{Punto de silla} \]

En \((1, 1)\): \(f_{xx} = 6\), \(f_{yy} = 6\), \(f_{xy} = -3\).

\[ D = (6)(6) - (-3)^2 = 36 - 9 = 27 > 0, \quad f_{xx} = 6 > 0 \implies \text{Mínimo local} \]

5.3 Herramienta de cálculo: Hessiano automático

A continuación se presenta una herramienta de cálculo simbólico que calcula automáticamente el Hessiano, lo evalúa en un punto crítico y clasifica el punto. El usuario puede modificar la función y el punto en el código para analizar sus propios ejemplos.

5.3.1 Ejemplo: \(f(x, y) = x^3 + y^3 - 3xy\) en \((1, 1)\)

library(Deriv)

# Defina aquí su función y el punto crítico
f <- function(x, y) x^3 + y^3 - 3*x*y
a <- 1  # coordenada x del punto crítico
b <- 1  # coordenada y del punto crítico

# Cálculo simbólico de derivadas
fx <- Deriv(f, "x")
fy <- Deriv(f, "y")
fxx <- Deriv(fx, "x")
fyy <- Deriv(fy, "y")
fxy <- Deriv(fx, "y")

# Evaluación en el punto (a, b)
H11 <- fxx(a, b)
H22 <- fyy(a, b)
H12 <- fxy(a, b)
D <- H11 * H22 - H12^2

# Clasificación
if (D > 0) {
  if (H11 > 0) clasif <- "MÍNIMO LOCAL"
  else if (H11 < 0) clasif <- "MÁXIMO LOCAL"
  else clasif <- "Indeterminado (D > 0 pero f_xx = 0)"
} else if (D < 0) {
  clasif <- "PUNTO DE SILLA"
} else {
  clasif <- "CRITERIO INCONCLUSO (D = 0)"
}

cat("=== ANÁLISIS DEL HESSIANO ===\n\n")
## === ANÁLISIS DEL HESSIANO ===
cat("Función: f(x,y) = x³ + y³ - 3xy\n")
## Función: f(x,y) = x³ + y³ - 3xy
cat("Punto crítico: (", a, ", ", b, ")\n\n", sep = "")
## Punto crítico: (1, 1)
cat("f_xx =", H11, "\n")
## f_xx = 6
cat("f_yy =", H22, "\n")
## f_yy = 6
cat("f_xy =", H12, "\n\n")
## f_xy = -3
cat("Discriminante D = f_xx·f_yy - (f_xy)² =", D, "\n\n")
## Discriminante D = f_xx·f_yy - (f_xy)² = 27
cat("CLASIFICACIÓN:", clasif, "\n")
## CLASIFICACIÓN: MÍNIMO LOCAL

5.3.2 Segundo ejemplo: \(f(x, y) = x^2 + y^2\) en \((0, 0)\)

f2 <- function(x, y) x^2 + y^2
a2 <- 0; b2 <- 0

fx2 <- Deriv(f2, "x")
fy2 <- Deriv(f2, "y")
fxx2 <- Deriv(fx2, "x")
fyy2 <- Deriv(fy2, "y")
fxy2 <- Deriv(fx2, "y")

H11_2 <- fxx2(a2, b2)
H22_2 <- fyy2(a2, b2)
H12_2 <- fxy2(a2, b2)
D2 <- H11_2 * H22_2 - H12_2^2

if (D2 > 0 && H11_2 > 0) clasif2 <- "MÍNIMO LOCAL"
cat("f(x,y) = x² + y² en (0,0): D =", D2, ", f_xx =", H11_2, "→", clasif2, "\n")
## f(x,y) = x² + y² en (0,0): D = 4 , f_xx = 2 → MÍNIMO LOCAL

5.4 Extremos absolutos en regiones compactas

El teorema de Weierstrass garantiza que una función continua sobre un conjunto cerrado y acotado (compacto) alcanza su máximo y mínimo absolutos.

5.4.1 Método para encontrar extremos absolutos

  1. Encuentre los puntos críticos en el interior de la región.
  2. Encuentre los extremos de \(f\) restringida a la frontera (usando parametrización o multiplicadores de Lagrange).
  3. Evalúe \(f\) en todos los puntos candidatos y compare.

5.4.2 Ejemplo resuelto

Ejemplo 2. Halle los extremos absolutos de \(f(x, y) = x^2 + y^2 - x - y\) en el disco \(x^2 + y^2 \leq 1\).

Solución. Interior: \(f_x = 2x - 1 = 0\), \(f_y = 2y - 1 = 0\). Punto crítico: \((\frac{1}{2}, \frac{1}{2})\).

\(f(\frac{1}{2}, \frac{1}{2}) = \frac{1}{4} + \frac{1}{4} - \frac{1}{2} - \frac{1}{2} = -\frac{1}{2}\).

Frontera: Parametrizamos \(x = \cos t\), \(y = \sin t\). Entonces:

\[ g(t) = 1 - \cos t - \sin t \]

\(g'(t) = \sin t - \cos t = 0 \implies \tan t = 1\), es decir, \(t = \pi/4\) o \(t = 5\pi/4\).

  • \(t = \pi/4\): \((\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2})\), \(f = 1 - \sqrt{2} \approx -0.414\)
  • \(t = 5\pi/4\): \((-\frac{\sqrt{2}}{2}, -\frac{\sqrt{2}}{2})\), \(f = 1 + \sqrt{2} \approx 2.414\)

Comparando: mínimo absoluto \(-\frac{1}{2}\) en \((\frac{1}{2}, \frac{1}{2})\), máximo absoluto \(1 + \sqrt{2}\) en \((-\frac{\sqrt{2}}{2}, -\frac{\sqrt{2}}{2})\).

5.5 Multiplicadores de Lagrange

El método de los multiplicadores de Lagrange permite encontrar extremos de una función \(f(x, y)\) sujeta a una restricción \(g(x, y) = 0\).

5.5.1 Condición necesaria

Si \(f\) tiene un extremo en \((a, b)\) sujeto a \(g(x, y) = 0\), y \(\nabla g(a, b) \neq \vec{0}\), entonces existe un escalar \(\lambda\) (llamado multiplicador de Lagrange) tal que:

\[ \nabla f(a, b) = \lambda \nabla g(a, b) \]

Esto, junto con la restricción \(g(a, b) = 0\), forma un sistema de ecuaciones.

5.5.2 Función de Lagrange

Definimos:

\[ \mathcal{L}(x, y, \lambda) = f(x, y) - \lambda g(x, y) \]

Los puntos críticos de \(\mathcal{L}\) son los candidatos a extremos condicionados.

5.5.3 Ejemplo resuelto

Ejemplo 3. Halle el punto de la recta \(x + y = 1\) más cercano al origen.

Solución. Queremos minimizar \(f(x, y) = x^2 + y^2\) sujeto a \(g(x, y) = x + y - 1 = 0\).

La función de Lagrange es:

\[ \mathcal{L} = x^2 + y^2 - \lambda(x + y - 1) \]

Las condiciones son:

\[ \begin{aligned} \mathcal{L}_x &= 2x - \lambda = 0 \\ \mathcal{L}_y &= 2y - \lambda = 0 \\ \mathcal{L}_\lambda &= -(x + y - 1) = 0 \end{aligned} \]

De las primeras dos ecuaciones: \(2x = 2y = \lambda \implies x = y\). Sustituyendo en la tercera: \(2x = 1 \implies x = y = \frac{1}{2}\).

El punto más cercano es \((\frac{1}{2}, \frac{1}{2})\) y la distancia mínima es \(\sqrt{\frac{1}{4} + \frac{1}{4}} = \frac{\sqrt{2}}{2}\).

5.6 Extensión a tres variables

Para una función \(f(x, y, z)\) sujeta a una restricción \(g(x, y, z) = 0\), la condición es análoga:

\[ \nabla f = \lambda \nabla g, \qquad g = 0 \]

Para dos restricciones \(g = 0\) y \(h = 0\):

\[ \nabla f = \lambda \nabla g + \mu \nabla h, \qquad g = 0, \quad h = 0 \]
