Conceptos de Superficies

Capítulo 4 Funciones de Varias Variables y Derivadas Parciales

🎧 Podcast del capítulo

El cálculo diferencial de funciones de varias variables es la base del análisis de superficies y campos escalares. En este capítulo se desarrollan las herramientas fundamentales: límites, continuidad, derivadas parciales, plano tangente, regla de la cadena, derivada direccional y gradiente.

4.1 Funciones de dos y tres variables

4.1.1 Definición

Una función de dos variables es una regla que asigna a cada par ordenado \((x, y)\) en un conjunto \(D \subseteq \mathbb{R}^2\) un único número real \(z = f(x, y)\).

De manera análoga, una función de tres variables asigna a cada terna \((x, y, z) \in D \subseteq \mathbb{R}^3\) un valor \(w = f(x, y, z)\).

4.1.2 Dominio y rango

El dominio de \(f\) es el conjunto de todos los puntos donde la función está definida. El rango es el conjunto de todos los valores posibles de \(f\).

4.1.3 Ejemplos

Función Dominio Rango
\(f(x, y) = x^2 + y^2\) \(\mathbb{R}^2\) \([0, \infty)\)
\(f(x, y) = \sqrt{1 - x^2 - y^2}\) \(x^2 + y^2 \leq 1\) \([0, 1]\)
\(f(x, y) = \frac{1}{x - y}\) \(x \neq y\) \(\mathbb{R} \setminus \{0\}\)
\(f(x, y, z) = x^2 + y^2 + z^2\) \(\mathbb{R}^3\) \([0, \infty)\)

4.2 Gráfica de una función de dos variables

La gráfica de \(z = f(x, y)\) es el conjunto de puntos \((x, y, z) \in \mathbb{R}^3\) tal que \(z = f(x, y)\). Geométricamente, es una superficie en el espacio tridimensional.

4.2.1 Ejemplo

La gráfica de \(f(x, y) = x^2 + y^2\) es un paraboloide elíptico que abre hacia arriba con vértice en el origen.

Figure 4.1: Paraboloide elíptico \(z = x^2 + y^2\).

4.3 Curvas de nivel

Las curvas de nivel de una función \(f(x, y)\) son las curvas en el plano \(xy\) definidas por \(f(x, y) = k\), donde \(k\) es una constante. Proporcionan una representación bidimensional de la función, análoga a los mapas topográficos.

4.3.1 Ejemplos de curvas de nivel

Función Curvas de nivel Descripción
\(f(x, y) = x^2 + y^2\) \(x^2 + y^2 = k\) Círculos concéntricos
\(f(x, y) = x^2 - y^2\) \(x^2 - y^2 = k\) Hipérbolas
\(f(x, y) = x + y\) \(x + y = k\) Rectas paralelas

Figure 4.2: Curvas de nivel del paraboloide \(z = x^2 + y^2\).

4.4 Límites y continuidad

4.4.1 Límite de una función de dos variables

Decimos que \(\lim_{(x,y) \to (a,b)} f(x, y) = L\) si para todo \(\varepsilon > 0\) existe \(\delta > 0\) tal que:

\[ 0 < \sqrt{(x-a)^2 + (y-b)^2} < \delta \implies |f(x, y) - L| < \varepsilon \]

La dificultad principal es que \((x, y)\) puede aproximarse a \((a, b)\) por infinitos caminos.

4.4.2 Criterio para inexistencia de límite

Si al aproximarnos por dos caminos distintos obtenemos valores límite diferentes, entonces el límite no existe.

4.4.3 Ejemplo resuelto

Ejemplo 1. Demuestre que \(\lim_{(x,y) \to (0,0)} \frac{xy}{x^2 + y^2}\) no existe.

Solución. Aproximando por el eje \(x\) (\(y = 0\)):

\[ \lim_{x \to 0} \frac{x \cdot 0}{x^2 + 0} = 0 \]

Aproximando por la recta \(y = x\):

\[ \lim_{x \to 0} \frac{x \cdot x}{x^2 + x^2} = \lim_{x \to 0} \frac{x^2}{2x^2} = \frac{1}{2} \]

Como \(0 \neq \frac{1}{2}\), el límite no existe.

4.4.4 Continuidad

Una función \(f\) es continua en \((a, b)\) si:

  1. \(f(a, b)\) está definida.
  2. \(\lim_{(x,y) \to (a,b)} f(x, y)\) existe.
  3. \(\lim_{(x,y) \to (a,b)} f(x, y) = f(a, b)\).

4.5 Derivadas parciales

4.5.1 Definición

La derivada parcial de \(f\) respecto a \(x\) en \((a, b)\) se define como:

\[ f_x(a, b) = \lim_{h \to 0} \frac{f(a + h, b) - f(a, b)}{h} \]

De manera análoga:

\[ f_y(a, b) = \lim_{h \to 0} \frac{f(a, b + h) - f(a, b)}{h} \]

En la práctica, calculamos \(f_x\) derivando \(f\) respecto a \(x\) tratando a \(y\) como constante, y viceversa.

4.5.2 Notaciones

\[ f_x = \frac{\partial f}{\partial x} = \partial_x f, \qquad f_y = \frac{\partial f}{\partial y} = \partial_y f \]

4.5.3 Interpretación geométrica

  • \(f_x(a, b)\) es la pendiente de la recta tangente a la curva de intersección de la superficie \(z = f(x, y)\) con el plano vertical \(y = b\).
  • \(f_y(a, b)\) es la pendiente de la recta tangente a la curva de intersección con el plano vertical \(x = a\).

4.5.4 Ejemplo resuelto

Ejemplo 2. Calcule las derivadas parciales de \(f(x, y) = x^3 y^2 + 2xy^4\).

Solución.

\[ f_x = 3x^2 y^2 + 2y^4 \]

\[ f_y = 2x^3 y + 8xy^3 \]

4.6 Herramienta de cálculo: Derivadas parciales paso a paso

A continuación se presenta una herramienta de cálculo simbólico que permite obtener las derivadas parciales de cualquier función polinómica. El usuario puede modificar la función en el código para calcular sus propias derivadas.

4.6.1 Ejemplo: \(f(x, y) = x^3 \sin(y) + e^{xy}\)

library(Deriv)

# Defina aquí su función
f <- function(x, y) x^3 * sin(y) + exp(x * y)

# Cálculo simbólico
fx <- Deriv(f, "x")
fy <- Deriv(f, "y")
fxx <- Deriv(fx, "x")
fyy <- Deriv(fy, "y")
fxy <- Deriv(fx, "y")

cat("Función original: f(x,y) = x³·sin(y) + e^(x·y)\n\n")
## Función original: f(x,y) = x³·sin(y) + e^(x·y)
cat("∂f/∂x =", as.character(body(fx)), "\n")
## ∂f/∂x = + 3 * (x^2 * sin(y)) y * exp(x * y)
cat("∂f/∂y =", as.character(body(fy)), "\n")
## ∂f/∂y = * x exp(x * y) + x^2 * cos(y)
cat("∂²f/∂x² =", as.character(body(fxx)), "\n")
## ∂²f/∂x² = + 6 * (x * sin(y)) y^2 * exp(x * y)
cat("∂²f/∂y² =", as.character(body(fyy)), "\n")
## ∂²f/∂y² = * x^2 exp(x * y) - x * sin(y)
cat("∂²f/∂x∂y =", as.character(body(fxy)), "\n")
## ∂²f/∂x∂y = { .e1 <- x * y (1 + .e1) * exp(.e1) + 3 * (x^2 * cos(y))

4.7 Derivadas parciales de orden superior

Las segundas derivadas parciales son:

\[ \begin{aligned} f_{xx} &= \frac{\partial^2 f}{\partial x^2} = \frac{\partial}{\partial x}\left(\frac{\partial f}{\partial x}\right) \\ f_{yy} &= \frac{\partial^2 f}{\partial y^2} = \frac{\partial}{\partial y}\left(\frac{\partial f}{\partial y}\right) \\ f_{xy} &= \frac{\partial^2 f}{\partial y \partial x} = \frac{\partial}{\partial y}\left(\frac{\partial f}{\partial x}\right) \\ f_{yx} &= \frac{\partial^2 f}{\partial x \partial y} = \frac{\partial}{\partial x}\left(\frac{\partial f}{\partial y}\right) \end{aligned} \]

4.7.1 Teorema de Clairaut

Si \(f_{xy}\) y \(f_{yx}\) son continuas en una región, entonces:

\[ f_{xy} = f_{yx} \]

Es decir, el orden de derivación no importa (bajo condiciones de regularidad).

4.8 Plano tangente y aproximación lineal

4.8.1 Ecuación del plano tangente

Si \(f\) es diferenciable en \((a, b)\), el plano tangente a la superficie \(z = f(x, y)\) en el punto \((a, b, f(a, b))\) tiene ecuación:

\[ z = f(a, b) + f_x(a, b)(x - a) + f_y(a, b)(y - b) \]

4.8.2 Aproximación lineal (diferenciabilidad)

La función \(f\) es diferenciable en \((a, b)\) si puede aproximarse linealmente cerca de ese punto:

\[ f(x, y) \approx f(a, b) + f_x(a, b)(x - a) + f_y(a, b)(y - b) \]

El error de esta aproximación tiende a cero más rápidamente que la distancia \(\sqrt{(x-a)^2 + (y-b)^2}\).

4.8.3 Ejemplo resuelto

Ejemplo 3. Halle la ecuación del plano tangente a \(f(x, y) = x^2 + y^2\) en \((1, 2)\).

Solución. \(f(1, 2) = 5\), \(f_x = 2x \implies f_x(1, 2) = 2\), \(f_y = 2y \implies f_y(1, 2) = 4\).

\[ z = 5 + 2(x - 1) + 4(y - 2) = 2x + 4y - 5 \]

4.9 Regla de la cadena

4.9.1 Caso 1: Función compuesta de una variable

Si \(z = f(x, y)\) donde \(x = x(t)\) e \(y = y(t)\), entonces:

\[ \frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt} \]

4.9.2 Caso 2: Función compuesta de dos variables

Si \(z = f(x, y)\) donde \(x = x(s, t)\) e \(y = y(s, t)\), entonces:

\[ \begin{aligned} \frac{\partial z}{\partial s} &= \frac{\partial f}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial s} \\ \frac{\partial z}{\partial t} &= \frac{\partial f}{\partial x}\frac{\partial x}{\partial t} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial t} \end{aligned} \]

4.9.3 Ejemplo resuelto

Ejemplo 4. Sea \(z = x^2 y\) donde \(x = \sin t\) e \(y = \cos t\). Calcule \(dz/dt\).

Solución.

\[ \frac{dz}{dt} = (2xy)(\cos t) + (x^2)(-\sin t) = 2\sin t \cos^2 t - \sin^3 t \]

4.10 Derivada direccional y gradiente

4.10.1 Derivada direccional

La derivada direccional de \(f\) en \((a, b)\) en la dirección del vector unitario \(\vec{u} = (u_1, u_2)\) mide la tasa de cambio de \(f\) en esa dirección:

\[ D_{\vec{u}} f(a, b) = \lim_{h \to 0} \frac{f(a + hu_1, b + hu_2) - f(a, b)}{h} \]

Si \(f\) es diferenciable, esta fórmula se simplifica a:

\[ D_{\vec{u}} f(a, b) = f_x(a, b) u_1 + f_y(a, b) u_2 \]

4.10.2 Vector gradiente

El gradiente de \(f\) es el vector de las derivadas parciales:

\[ \nabla f = (f_x, f_y) \]

En tres variables:

\[ \nabla f = (f_x, f_y, f_z) \]

La derivada direccional puede escribirse como:

\[ D_{\vec{u}} f = \nabla f \cdot \vec{u} \]

4.10.3 Propiedades del gradiente

  1. Dirección de máximo crecimiento: el gradiente apunta en la dirección de máxima tasa de crecimiento de \(f\).
  2. Magnitud del máximo crecimiento: \(\|\nabla f\|\) es la máxima tasa de crecimiento.
  3. Perpendicularidad: el gradiente es perpendicular a las curvas (o superficies) de nivel.

4.10.4 Ejemplo resuelto

Ejemplo 5. Sea \(f(x, y) = x^2 + y^2\). Calcule el gradiente en \((1, 2)\), la derivada direccional en la dirección \(\vec{u} = (\frac{3}{5}, \frac{4}{5})\), y la dirección de máximo crecimiento.

Solución. \(\nabla f = (2x, 2y)\), por tanto \(\nabla f(1, 2) = (2, 4)\).

Derivada direccional:

\[ D_{\vec{u}} f(1, 2) = (2, 4) \cdot \left(\frac{3}{5}, \frac{4}{5}\right) = \frac{6}{5} + \frac{16}{5} = \frac{22}{5} \]

Dirección de máximo crecimiento: \(\nabla f(1, 2) = (2, 4)\), o normalizado: \(\frac{(2, 4)}{\sqrt{20}} = \frac{(1, 2)}{\sqrt{5}}\).

Máxima tasa de crecimiento: \(\|\nabla f(1, 2)\| = \sqrt{4 + 16} = \sqrt{20} = 2\sqrt{5}\).
